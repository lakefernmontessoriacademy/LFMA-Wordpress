Backup Database for WordPress is the best backup choice for WordPress based websites or blogs.

== Description ==
We offer the easiest way to <strong>backup</strong> your WordPress based website or blog. You can backup your database just on click.


== Installation ==

Installation of Database Backup

1. Upload `Backup Database` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Tools->Backup Database
1. Enjoy!
